## 🚀 安装说明

### 方式一：开发者模式安装（推荐）

1. **下载项目**

   ```bash
   # 克隆仓库
   git clone https://github.com/yourusername/super_resume.git

   # 或直接下载ZIP文件并解压
   ```

2. **打开 Chrome 扩展管理页面**

   - 在 Chrome 浏览器地址栏输入：`chrome://extensions/`
   - 或通过菜单：`更多工具` → `扩展程序`

3. **启用开发者模式**

   - 点击页面右上角的"开发者模式"开关

4. **加载扩展**

   - 点击"加载已解压的扩展程序"
   - 选择项目根目录（包含 manifest.json 的文件夹）
   - 确认加载

5. **验证安装**
   - 在扩展列表中看到"简历自动填写助手"图标
   - 点击图标可以打开弹出窗口
   - 查看版本号确认为 v2.0

## 📖 使用指南

### 快速开始

#### 第一步：打开扩展

1. 点击浏览器工具栏的扩展图标
2. 或使用快捷键（可在扩展管理中设置）
3. 弹出窗口显示"手动填写"界面

#### 第二步：准备简历数据

**方式 A：上传简历文件**

1. 在"手动填写"标签页找到"智能上传简历"区域
2. 拖拽简历文件到上传区域，或点击选择文件
3. 支持格式：PDF、DOCX、JSON、TXT
4. 等待解析完成（显示绿色成功提示）
5. 点击"使用解析数据"按钮填充表单
6. 检查并完善自动填充的信息

**方式 B：手动输入**

1. 在各个模块中填写相应信息
2. 使用"+ 添加"按钮添加多条经历
3. 使用"删除"按钮移除不需要的项
4. 系统会自动保存您的输入

#### 第三步：自动填写招聘网站

1. 打开目标招聘网站的简历填写页面
   - 如：智联招聘、前程无忧、Boss 直聘等
2. 点击扩展图标打开弹出窗口
3. 确认简历信息已完善
4. 点击"自动填写页面"按钮
5. 等待填充完成（显示成功提示）
6. 检查自动填充的内容，手动调整个别字段
